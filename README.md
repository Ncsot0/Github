📝 Mis Tareas - Técnico en Diseño y Desarrollo de Software

Este repositorio está destinado exclusivamente a la carga y organización de las tareas académicas realizadas durante el cursado de la carrera. Todos los ejercicios han sido desarrollados y testeados utilizando Visual Studio Code.
🛠️ Entorno de Desarrollo

Para la resolución de estas tareas, utilizo un entorno basado en:

    Editor: Visual Studio Code (VS Code).

    Control de Versiones: Git & GitHub.

    Extensiones clave: Python (Pylance), Django, ES7+ React/Redux/GraphQL/React-Native snippets, y herramientas de formateo de CSS/HTML.

📚 Tecnologías Aplicadas

Las tareas subidas abarcan los siguientes lenguajes y frameworks solicitados en la carrera:

    Lenguajes: Python, JavaScript, HTML5, CSS3.

    Backend: Django, Node.js.

    Bases de Datos: SQL y NoSQL.

    Herramientas: Gestión de repositorios con Git.
